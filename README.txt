== Installation ==

1. In your WordPress dashboard, go to Appearance > Themes.
2. Click the "Add New" button and select "Upload Theme."
3. Upload the zip file containing your theme.
4. Click "Install Now" to install the theme.
5. Click "Activate" to activate the theme.
